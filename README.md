# Requirements
- Java 11
- Gradle 7.6.1

# Installation
- Build `./gradlew build`
- Run `./gradlew run`

# API Request
- Method: POST
- URL: localhost:7000/ocr?filename=%s. ex: `localhost:7000/ocr?filename=ImageWithWords1.jpg`
